## 路線定案標準

- 音質可接受（無明顯外國腔、zh-TW 穩定不飄簡中發音）**且**工具情境轉接成功率 100% → 採 Live API proxy 路線。
- 音質不可接受，或 AUDIO 輸出在 zh-TW 不穩定 → 走自建管線（規劃文件 Phase 1–4），WS gateway 沿用。
- 延遲僅記錄供參（POC 寬鬆標準：首音 p50 < 1.5s），不單獨作為否決條件。
